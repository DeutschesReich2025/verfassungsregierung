# Verfassungsregierung des Archivstaates

Diese Seite dokumentiert die souveräne Verfassungsregierung nach Artikel 1–50 der Grundordnung.  
Sie dient der Archivierung, Sichtbarmachung und dem Schutz der verfassungsgebundenen Mandatsstruktur.

## Inhalte
- Verfassung
- Staatssymbole (Flagge, Siegel, Logo)
- Mandatsstruktur
- Zeitachsen und Archivmodule
